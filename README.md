# Personal Portfolio Website

## Project Overview

This project is a Personal Portfolio Website developed as part of the **HTML5 Semantic Structure & Accessibility** task. The website demonstrates the use of semantic HTML5 elements, accessibility best practices, and responsive design principles.

## Features

- Semantic HTML5 structure
- Accessible navigation
- SEO-friendly meta tags
- About Me section
- Projects section
- Contact form
- GitHub profile integration
- Responsive layout

## Technologies Used

- HTML5
- CSS3

## Project Structure

```
portfolio-task01
│
├── index.html
├── css
│   └── style.css
└── README.md
```

## Semantic Elements Used

- `<header>`
- `<nav>`
- `<main>`
- `<section>`
- `<article>`
- `<footer>`

## Accessibility Features

- ARIA labels for navigation
- Proper heading hierarchy
- Keyboard-friendly navigation
- Form labels for all inputs
- Responsive viewport settings

1. Download or clone the repository.

